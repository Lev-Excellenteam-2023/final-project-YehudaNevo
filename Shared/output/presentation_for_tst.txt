['PE format structure and explanation', 'The Portable Executable (PE) format is the file format used for executable files, object code, and DLLs (dynamic-link libraries) in Windows. It contains information on the structure of the file, including its header, sections, and other data. As a result, it lets Windows know everything necessary to load, execute, and manage the code in a cohesive way.']

['IDA -  tool for reverse exe file', 'IDA (Interactive Disassembler) is a software tool used for reverse engineering executable files. It allows users to analyze, understand, and modify binary code at a low-level. IDA can be used to help identify vulnerabilities, create exploits, and debug code.']

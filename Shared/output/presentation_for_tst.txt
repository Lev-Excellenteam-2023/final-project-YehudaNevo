['PE format structure and explanation', 'The Portable Executable (PE) format is a file format used for executables, object code, and dynamic-link libraries (DLLs) in Windows operating systems. The format consists of a header followed by a series of sections that contain executable code and data. The header includes essential information such as entry point address, image file size, and subsystem type.']

Slide 2:
['IDA -  tool for reverse exe file', 'IDA (Interactive Disassembler) is a software application used for reverse engineering of binary files. It is capable of disassembling machine code and generating the code in assembly language format, making it easier for developers to understand how a program works. IDA is a commonly used tool in the field of software security and vulnerability research.']

['PE format structure and explanation', 'PE format is a file format used by Windows for storing executable code, DLLs, and other binary files. It contains a header, followed by sections that hold the code, data, and resources. The header contains important information about the binary, such as the entry point address and linker information.']

Slide 2:
['IDA -  tool for reverse exe file', 'IDA (Interactive Disassembler) is a software tool used to analyze and reverse engineer binary files, including executable files. It allows you to inspect the low-level machine code of the file and understand its inner workings, making it a powerful tool for reverse engineering and understanding how software operates. IDA is widely used by security professionals and software engineers.']
